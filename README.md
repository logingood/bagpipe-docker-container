# bagpipe-docker-container
Dockerfile to create [BaGPipe BGP](https://github.com/Orange-OpenSource/bagpipe-bgp) Docker container

# Install
1. Clone repository
```
git clone https://github.com/murat1985/bagpipe-docker-container
```
2. Run build
```
docker build .
```
3. Execut start.sh
```
./start.sh
```
